mob6
====

Hierbij het idee/concept voor mijn mobiele applicatie.

Ik wil een mobile app maken waarmee ik de festivals van Nederland in kaart breng. De festivals zullen op locatie in een google map kaart van Nederland geplaatst worden door middel van een marker (logo/naam van het evenement). De gebruiker kan hierop klikken waardoor er een dialoog venster komt om meer informatie te verkrijgen.

De gegevens die in een dialoog venster komen te staan zijn:
        Naam festival
        Datum/tijd
        Locatie
        Korte omschrijving
        Line-up van artiesten
        Impressie video
        Prijs van een kaartje (ticket service link?)
        Facebook like button

De ontwikkel omgeving word phonegap met een database er achter. De sensoren/api  van de telefoon die gebruikt zullen worden zijn de notification en geolocation. De externe api is Googlemaps. De app word uitgebracht voor android toestellen.

Dit word de basis van mijn mobiele applicatie.
